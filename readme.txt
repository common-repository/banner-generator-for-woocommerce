By this plugin, you can build and manage animated gif banners with your woocommerce products.
== Description ==

The Features:
Easy to use,
Generate unlimited banners,
Rich settings to customize,
Generated banners are easy to publish.

Requirements:
WooCommerce 2.3 or above
PHP 5.5 or above
WordPress 4.0 or above



== Installation ==

= From your WordPress dashboard =
1. Visit 'Plugins > Add New'
2. Search for 'Banner Generator For Woocommerce'
3. Activate plugin from your Plugins page.
4. Go to  'Banner Generator > Manage banners' for create the banner.
5. If you have pretty permalinks enabled, visit yourdomain.com/forums, or if you do not have pretty permalinks enabled, visit yourdomain.com?post_type=forum

= From WordPress.org =
1. Unzip the downloaded zip file.
2. Upload banner-generator-for-woocommerce folder to the /wp-content/plugins/ directory
3. Activate the plugin through the \'Plugins\' menu in WordPress
4. Go to  'Banner Generator > Manage banners' for create the banner.
